## The Address Book Application

### Use Case

Pascal Pausenkurt uses an address book application. After choosing a name the adress book prints out the corresponding address. 
### Implementation
* the application finds during one run only one address
* the addresses are:
  * Harry lives Middle Of 9, Nowhere
  * Meghan lives Ebony Tower, Far Away
  * William and Kate live Dungeon Way 7, Kent
  * John and Georgia live in Mercy Road 12, Liverpool
  * Paula and Richard live in Abbey Road, London
  
### Hint
* it seems that the application focusses on dictionary, but there is a good reason to use arrays as well

### Definition of Done 

* application can be started and prompts for a user input
* after entering a name the appropriate adress information is printed to the console
