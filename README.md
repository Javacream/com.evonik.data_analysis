# com.evonik.dataprocessor.level2

* directory names correspond to path and unit nr
* markdown file names:
  * todo
    * guided training lessons
    * questions
  * lessionslearned
    * reached goals
  * ressources
    * list of online tutorials, videos or handouts
  * exercise
    * training labs with use case and definition of done
  * workshop
    * set of ideas, scratches that should be processed during a moderated group session, workshop, bar camp
  * story
    * kick of to write a user story as training introduction / motivation
